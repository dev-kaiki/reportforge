# ReportForge

Reports + history + export (PDF/JSON)

## Links
- Live (Vercel): https://dev-kaiki-reportforge.vercel.app
- Swagger (Render): https://dev-kaiki-reportforge-api.onrender.com/docs
- Health: https://dev-kaiki-reportforge-api.onrender.com/health

## One-click deploy

API (Render Blueprint):
https://render.com/deploy?repo=https://github.com/dev-kaiki/reportforge

WEB (Vercel):
https://vercel.com/new/clone?repository-url=https://github.com/dev-kaiki/reportforge&project-name=dev-kaiki-reportforge&repository-name=reportforge&root-directory=apps/web

Vercel ENV:
NEXT_PUBLIC_API_URL=https://dev-kaiki-reportforge-api.onrender.com

## Local (Windows)
Run:
powershell -ExecutionPolicy Bypass -File .\dev.ps1
